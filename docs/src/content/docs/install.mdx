Sprout is available as a small, single binary on both Mac and Linux. Installation commands are below, or you can [download a release from GitHub](https://github.com/talss89/sprout/releases).

<Tabs>
  <TabItem label="Mac">
    ```
    brew install sprout
    ```
</TabItem>
  <TabItem label="Linux">
     ```
     brew install sprout
     ```
  </TabItem>
</Tabs>
